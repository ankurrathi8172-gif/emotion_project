# Facial Emotion Recognition with Dashboard & Report Generator

## Features
- Detects facial emotion using webcam
- Stores detected emotions with timestamp in CSV
- Dashboard for emotion analytics
- Bar graph and pie chart visualization
- Downloadable report

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run emotion detection:
   python main.py

3. Run dashboard:
   streamlit run dashboard.py
